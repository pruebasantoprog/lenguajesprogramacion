POKÉAPI

Esta página web utiliza la API RESTful de Pokémon PokeApi.

Para obtener la imagen de Pikachu en movimiento se usa un fetch para obtener la imagen del archivo JSON que proporciona la API. Cuando buscamos un Pokemon se llama a una función usando Async/await que obtiene la imagen y las caracteristicas del archivo JSON y se los pasa a otra función que crea el efecto de barra de progreso. Nombres de pokemon conocidos: Pikachu, Charizard, Mewtwo, Eevee, Gengar, Mew, Snorlax, Bulbasaur, etc.
