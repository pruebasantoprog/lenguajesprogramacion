Ejemplos JavaScript

jQuery

jQuery

jQuery es una librería o biblioteca multiplataforma de JavaScript de código abierto y tiene como finalidad principal interactuar con documentos HTML, manipular el DOM(Modelo de Objetos del Documento), controlar eventos, implementar animaciones e interacción AJAX (JavaScript asíncrono y XML)

Async/Await

Async/Await

Las funciones Async/Await facilitan la gestión de tareas que requieren tiempo, como esperar una respuesta de una API. Async y await son la alternativa a las promesas y los callbacks. Como muestra se usa en este ejemplo la API RESTful de Pokémon PokeApi.

Fetch API

Fetch API

Fetch es una API de JavaScript que permite hacer peticiones HTTP para obtener recursos de forma asíncrona desde un servidor. Ofrece una forma moderna, potente y flexible de realizar solicitudes HTTP desde navegadores web.
